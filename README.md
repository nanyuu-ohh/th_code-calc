~~这是一个轻量级的网页工具，主要用于泰国项目在概念/方案设计阶段快速评估建筑及各专业的强制性合规指标，减少前期频繁作业。~~
🚀 如何使用在线访问：无需安装任何环境，直接通过浏览器（推荐 Chrome/Edge）访问网页版即可使用：👉 点击这里访问速查器 (完美支持电脑和手机端适配，随时随地可用)
  -输入参数：项目定位：选择所在地（曼谷与外府的停车等指标有差异）及是否位于 IEAT 工业园。
  -建筑功能：支持多选（如综合体可选商业+办公+酒店）。核心数据：填入高度、面积、层数及深层地下室参数（将自动判定是否触发“高层”或“超大型”建筑强制条款）。
  -辅助数据：根据所选功能，按需填入人数、客房数或座位数，用于精准计算洁具数量。
  -查看报告：点击“生成”后，右侧（移动端为滑出排版）将自动分类输出建筑、结构、暖通、给排水、电气、电梯 6 大专业的指标界限及规范出处。
💡 核心功能与注意事项自动判定建筑级别：输入高度和面积后，系统会自动判定并高亮显示适用的核心法规（如 MR33、MR55）。
  -内置 MR63 洁具算法：自动计算马桶、小便斗、洗手盆及淋浴数量。页面底部带有可折叠的完整数据字典供人工核对。
  -混合功能测算局限：目前对于多选混合功能（如同时包含住宅+办公），系统的垃圾产出和部分洁具是基于“总面积”全局叠加估算。如需极其精准的单项指标，建议针对各功能区“分开填写、单独查算”。
  -原文追溯：各指标卡片右下角附有具体的法条编号（如 MR33 ข้อ22），页面底部提供泰国暹罗建筑师协会（ASA）的官方原文检索库链接。
*⚠️ 免责声明（重要）*
  -本工具数据截至 2026-03，抽取了部分核心控规条款。
  -结构及机电专业的提示仅为跨专业强制性底线条件，并非完整合规结果。
  -工具仅供设计前阶段概算参考，不可替代专业审图。
  -施工图及报建仍需以当地 LDI (合作院) 的最终核审意见为准。
